OPTAR (OPTical ARchiver)

*WebAssembly used, no server side processing. Files are not uploaded to any servers, preserving your privacy.

Backup any file with printable paper copies (print in A4, 600dpi on a laser printer). Note that file sizes are not preserved and there is often a small (~1-10%) irreparable error rate, so use archive formats (like rar with recovery records or par2 for builtin parity) or files where partial recovery is useful (such as txt or ogg video files). Originally made by Twibright Labs.

Use unoptar to decode scanned optar PNGs
